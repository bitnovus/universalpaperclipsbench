Universal Paperclips Leaderboard

Created by Frank Lantz, Universal Paperclips is a compact browser game that turns a single button into hours of compounding decisions, phase changes, and delayed payoff. That makes it a natural Harbor eval for long-running, long-horizon agents: success requires sustained browser control, adaptation from visible feedback, and a real prestige reset verified outside the agent context.

Methodology

Scoring

Reward is 1 only after the verifier sees hidden prestige state meet the target. The default target is one real prestige reset.

Isolation

The agent cannot read the game source, browser profile, verifier token, or hidden browser storage.

Controls

Agents use visible browser interaction through the CLI or the restricted MCP surface listed by each version.
